# Create VM with Arm Template


## 사용법
<b>깡통 VM 만들기</b>
<ol>
    <li>Azure Portal > [ + 리소스 만들기 ] > Template 검색 > Arm Template 만들기<li>
    <li>template.json 파일 복사 > 붙여넣기</li>
    <li>알맞은 변수를 입력하고 만들기</li>
</ol>

<b>VM 이미지에서 VM 만들기</b>
<ol>
    <li>Azure Portal > [ + 리소스 만들기 ] > Template 검색 > Arm Template 만들기<li>
    <li>template_from_img.json 파일 복사 > 붙여넣기</li>
    <li>알맞은 변수를 입력하고 만들기</li>
</ol>
